# Anti-counterfeiting-system
一物一码、防伪码查询系统，支持直接查询跟链接扫码跳转查询
采用PHP7.4+JS+CSS+Mysql5.7进行编写
滚动条采用simplebar  二维码生成采用qrcode框架 部分js代码采用jquery-3.6.0框架 图表绘制采用chart.js
拥有完整的权限设置，可以注册员工账号分配不同权限
